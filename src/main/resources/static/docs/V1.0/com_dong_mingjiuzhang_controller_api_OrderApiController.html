提交订单

作者: caishaodong

请求URL

/order/save POST

请求体 application/json

{
	"sendType":"int //取件方式：1自取  2邮寄（必传）",
	"receiverName":"string //收件人姓名（取件方式为邮寄的时候传递）",
	"receiverMobile":"string //收件人电话（取件方式为邮寄的时候传递）",
	"provinceCode":"int //省编码（取件方式为邮寄的时候传递）",
	"address":"string //详细地址（取件方式为邮寄的时候传递）",
	"payType":"int //购买类型：1课程 2：系列（必传）",
	"courseId":"long //课程id（购买类型为课程的时候传递）",
	"courseSeriesId":"long //课程系列id（购买类型为系列的时候传递）",
	"isGroup":"int //是否是团购：0否 1是（必传）"
}

返回结果

{
	"code":"int",
	"message":"string",
	"data":{}
}
